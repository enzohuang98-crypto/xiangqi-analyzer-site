快速開始

第一次使用

引擎與 AI 服務都由使用者自行選擇。軟體不提供引擎授權，也不包含 AI 額度。

  1. 1

    安裝 Windows App

    正式開放後，請從本網站下載已標示版本與 SHA-256 的安裝檔。

  2. 2

    加入本機象棋引擎

    在設定中選擇可信來源的引擎執行檔，測試連線後設為主要引擎。

  3. 3

    選擇性設定 AI

    貼上你自己的 API Key。只有使用 AI 中文解釋時才會呼叫對應服務並產生費用。

  4. 4

    擺局或匯入局面

    從棋盤、FEN 或棋譜建立局面，點選想走的棋後查看即時分析。

引擎與 AI

不綁定單一服務

實際相容性仍依引擎版本、協定實作與模型供應商政策而定。

本機引擎

UCI / UCCI

內建 Pikafish、象棋名手、象棋旋風、象棋小蟲、阿爾法貓與自訂引擎設定檔。

  • 引擎程式不隨安裝檔散布
  • 可設定主要與複核引擎
  • 執行前測試協定與連線狀態
雲端 AI

Claude / OpenAI / Gemini

AI 只負責解釋結構化引擎資料，不取代引擎做棋力判斷。

  • 使用者自行申請 API Key
  • 費用由各 Provider 直接計價
  • 可只用本機引擎、不啟用 AI
